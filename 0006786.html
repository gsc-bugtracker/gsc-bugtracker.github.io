(1) ножи

(4) гранатомёты

Дополнительная информация
1. The weapon is consider to be of 5 sorts (arguable, but it seems to me 2 won’t be enough)
(1) knives
(2) pistols
(3) automatic
(4) grenade cup discharge
(5) hand grenades
It is not fundamentally for me the weapon quantity of different kinds, so you have to decide solely how many it should be.

All rules are applied for each weapon type separately.

1 Rule

If NPC spawned something in inventory, it is not analyzing to throw it away, these items may be considered in other situations. So if you spawn two similar machineguns in NPC inventory, it won’t be immediately thrown off, in case he found a better one-a 2nd rule is applied, it throws both machine guns away

2 Rule

If NPC sees another weapon example:

First point: In case it is better then he has then:

- It picks it up
- It throws away all ammunition, fit to this weapon type only

Second point: weapon choice criteria

If it is not enough ammunition for new weapon - old one remains
Otherwise
If the new weapon is less used- it is chosen
Otherwise
If the old weapon is less used- it is chosen
Otherwise (i.e. the same deterioration)
It is chosen a weapon which is more valuable, according to the Koan’s adjusted scale (there’s simple hierarchy), and otherwise, it is selected

Otherwise
If it is not enough ammunition for old weapon – a new one is selected
Otherwise
If the new weapon is less used- it is chosen
Otherwise
If the old weapon is less used- it is chosen
Otherwise (i.e. the same deterioration)
It is chosen a weapon which is more valuable, according to the Koan’s adjusted scale (there’s simple hierarchy), and otherwise, it is selected

Third point: ammunition sufficiency criteria

As we have almost every NPC with unlimited ammunition, it is consider to have enough ammunition if you have not empty (empty cannot be) ammunition set, since it provides its infinity.
